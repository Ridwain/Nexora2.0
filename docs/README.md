# Nexora Android 2

Fresh Android client for the Nexora 2.0 architecture.

Implemented sections:

- Section 1: runnable Compose scaffold.
- Section 2: Supabase database foundation.
- Section 3: backend RPC layer.
- Section 4: Android Supabase client foundation.
- Section 5: auth UI, OTP signup verification, pending OTP recovery, and auth navigation.
- Section 6: owner-first company onboarding and owner context creation.
- Section 7: tenant CRM shell and mobile-first workspace UI foundation.
- Section 8: tenant-scoped CRM contacts foundation.
- Section 9: contact detail, edit, and archive.
- Section 10: contact live search, filters, and sorting.
